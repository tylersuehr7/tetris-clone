# TetrisCXX

A re-implementation of the Tetris video game in C++ 20.

## Dependencies
- C++ 20
- `raylib`
- `cmake`
- `make`
- `git`
- `clang++` or `g++`

## Build Commands
- `$ make` will setup cmake project, build directory, and compile project.
- `$ make run` will run the project.
- `$ make clean` will remove build directory and binary.
